# 总结

如果了解了 vue create 整个过程，那么相信你对 vue add 命令的代码看起来应该比较轻松，因为 vue add 相当于只是 vue create 中的一部分，vue create 包含了插件的安装以及调用，vue add 命令只是将此功能分离了出来。

有个消息就是，接下来分析的 vue invoke 命令其实是 vue add 命名去掉了插件包安装的那一部分，所以同学们都懂噻😁😁。
